My Python Projects
These are the first projects I built while learning Python. I made them myself to practice what I was learning-loops, if/else, functions, and handling user input.
To-Do List-lets you add and view tasks. I used a list and a menu loop. This was my second project and the first time I used .append().
Number Guessing Game-the computer picks a number 1–100 and you guess until you get it, with "too high/too low" hints. My first real project-I learned how while loops and if/elif/else work here.
Bank Balance Program-check balance, deposit, and withdraw money. This was the first time I used functions and try/except for handling bad input. It was the hardest one so far.
Book Contact- A program to add contacts, view all contacts, search for a contact, and delete a contact. It uses a dictionary to store each name with its number. This was the most challenging project for me so far-especially getting the indentation right and learning how dictionaries work. 
What I learned: I started Python after learning some C++, so I was familiar with basic programming but still new to Python. I built these step by step, fixing a lot of errors along the way (indentation got me a lot!). I'm still learning and plan to add more projects.

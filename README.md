# Python Beginner Projects

This repository contains three beginner Python command-line projects.  
These projects were made to practice Python basics such as loops, conditions, functions, lists, user input, and error handling.

## Projects

### 1. To do List App

A simple command-line todo list program where the user can add and view tasks.

#### Features
- Add a task
- View all tasks
- Quit the program
- Handles invalid menu choices

#### Skills Used
- Lists
- While loops
- If/elif/else statements
- User input

#### Run

```bash
python todo.py
```

---

### 2. Number Guessing Game

A simple number guessing game where the program randomly chooses a number from 1 to 100, and the user tries to guess it.

#### Features
- Generates a random number
- Lets the user guess until correct
- Tells the user if the guess is too low or too high

#### Skills Used
- Random module
- While loops
- If/elif/else statements
- User input

#### Run

```bash
python guessing_game.py
```

---

### 3. Bank Balance Program

A simple command-line banking program where the user can check balance, deposit money, and withdraw money.

#### Features
- Check current balance
- Deposit money
- Withdraw money
- Quit the program
- Handles invalid number inputs

#### Skills Used
- Functions
- While loops
- If/elif/else statements
- Try/except error handling
- User input

#### Run

```bash
python bank_balance.py
```

Through these projects, I practiced:
